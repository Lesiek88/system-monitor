## System Monitor Dashboard
A full-stack project that collects real-time system metrics (CPU, RAM, swap, and disk usage) and displays them on a live-updating dashboard with charts and alerts.

## Tech Stack
- Python (FastAPI, psutil, MySQL)
- TypeScript
- React

## What I Learned
In this project, I learned how to build a REST API with FastAPI and background tasks in Python, and how to use TypeScript and React to fetch live data, manage state, and render a real-time chart.
